# Rent Splitter Python Project

This is a simple Python program that calculates rent, food, and electricity bills and splits the total among people.

## Features
- Takes user input
- Calculates total expenses
- Splits cost per person
- Includes exception handling

## How to Run
Run the Python file and enter the required inputs.
